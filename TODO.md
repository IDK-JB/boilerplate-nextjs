# Technical Stack: High-Performance Boilerplate

## Contains

### Authentication & Security

* **Clerk** (MFA, Social Auth, Passkeys, User Impersonation)
* **Arcjet** (Bot protection & Application security)

### Database & ORM

* **DrizzleORM** (Multi-dialect type-safe ORM)
* **PGlite** (Offline local development database)
* **Prisma Postgres** (Production managed database)
* **Drizzle Kit / Studio** (CLI migrations & GUI database explorer)

### Architecture & Code Quality

* **Zod** (Schema-based data validation)
* **React Hook Form** (Form management)
* **Prettier** (Code formatting)
* **Lefthook** (Git hooks management)
* **Commitlint / Commitizen** (Commit message standardization)
* **Knip** (Unused files & dependencies detection)

### Testing & Observability

* **Vitest** (Unit testing & Browser mode)
* **Playwright** (E2E & Integration testing)
* **Sentry / Spotlight** (Error monitoring & local debugging)
* **Codecov** (Code coverage reporting)
* **LogTape / Better Stack** (Structured logging & log management)
* **Checkly** (Monitoring as Code)
* **PostHog** (Product analytics)

### Tooling & DevOps

* **Storybook** (Isolated UI component development)
* **GitHub Actions** (CI/CD for testing & deployment)
* **CodeRabbit** (AI-powered code reviews)
* **Semantic Release** (Automated changelog & versioning)
* **Dependabot** (Automated dependency updates)
* **@ Prefix** (Absolute imports)

### Performance & SEO

* **Bundler Analyzer** (Package size optimization)
* **Lighthouse** (Performance & SEO score maximization)
* **Metadata API** (Dynamic SEO, JSON-LD, and Open Graph tags)
* **Sitemap / Robots.txt** (Search engine indexing)
